This directory contains the codes and auxiliary files to the research paper: Experimental study on the effects of tube spacing and fin structure on falling film mode transitions with three-dimensional finned tubes. 
Date: 08/09/2022, paper submitted to International Journal of Multiphase Flow

Authors: Jingdong Chen; Zheming Gao


Attention: You need to install a Python environment to compile the .py file.

Instruction:
1. Open the SVR_prediction.py
2. Customize the inputs: (Ga_25, s_div_xi, F), where Ga_25 = Ga^0.25, s_div_xi = s/xi.
3. Select a correct model.
4. Run the codes and the predicted Re value will be printed.
